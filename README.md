📱 Peer-to-Peer Marketplace App

A Flutter-based marketplace application that enables users to buy and sell second-hand goods بسهولة and securely. This app is designed to solve common challenges in peer-to-peer trading such as lack of trust, poor product discovery, and inefficient communication.

🚀 Overview

The Peer-to-Peer Marketplace App allows users to:

List products with images, descriptions, and pricing
Browse and search for items بسهولة
Chat directly with sellers داخل التطبيق
Rate and review other users
Make secure payments
Track orders in real-time

The app also supports offline browsing with cached data and automatically syncs when the user is back online.

🎯 Problem Statement

At TradeHub Platforms, users face several challenges when buying and selling second-hand goods:

❌ Lack of trust between buyers and sellers
❌ Poor product discovery and search experience
❌ Inefficient communication channels
❌ Limited access to secure payment options

This app aims to solve all of the above.
